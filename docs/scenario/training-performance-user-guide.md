# Training + Performance Documentation Index

Bu paket `Training Needs` və `Performance Evaluation` modullarını ilk dəfə istifadə edən şəxsin oxuyaraq işi başa düşməsi üçün hazırlanmış sənəd dəstidir.

## Oxu sırası

1. `Training Needs` istifadəçi bələdçisi  
   Bax: `/docs/scenario/training-needs-user-guide.md`
2. `Performance Evaluation` istifadəçi bələdçisi  
   Bax: `/docs/scenario/performance-evaluation-user-guide.md`
3. Admin / Operations guide  
   Bax: `/docs/scenario/training-performance-admin-ops-guide.md`

## Bu paket nəyi izah edir

- hər modulun ümumi məntiqi
- tab, kart və əsas komponentlərin rolu
- hər əsas formun field-ləri
- save olunanda nə baş verdiyi
- hansı məlumatın hansı modula və ya sonrakı mərhələyə təsir etdiyi
- düzgün əməliyyat sırası
- tipik HR, manager və reviewer ssenariləri

## İstifadə qaydası

- modulu ilk dəfə açırsansa əvvəl user guide oxu
- gündəlik idarəetmə və nəzarət üçün admin/ops guide-a keç
- Training və Performance bir-birinə bağlı olduğuna görə zəif nəticə və training need axınını birlikdə düşün

# Təlim və performans istifadəçi bələdçisi

Bu bələdçi iki bölməni birlikdə başa düşmək üçündür:

- `Təlim ehtiyacı`
- `Performans qiymətləndirməsi`

## Bu iki bölmə bir-biri ilə necə bağlıdır?
Sadə şəkildə:

1. Əvvəl qiymətləndirmə aparılır.
2. Zəif qalan sahələr görünür.
3. Sonra bu sahələr üzrə təlim planlana bilər.

Yəni performans nəticəsi sonrakı inkişaf işinə kömək edir.

## Hansı işi hansı bölmədə görmək lazımdır?

### Performans qiymətləndirməsi
Burada:
- qiymətləndirmə dövrü yaradılır
- forma və ya test hazırlanır
- qiymətləndirmə verilir
- nəticə görünür

### Təlim ehtiyacı
Burada:
- təlim ehtiyacı daxil edilir
- plan hazırlanır
- sessiya yaradılır
- nəticə izlənir

## İstifadə üçün ən rahat sıra
1. Əvvəl performans nəticəsinə baxın.
2. Zəif qalan sahəni müəyyən edin.
3. Sonra təlim ehtiyacı bölməsinə keçin.
4. Həmin ehtiyaca uyğun plan qurun.
5. Sessiya keçirildikdən sonra nəticəni yenidən izləyin.

## Bu bələdçi kim üçündür?
- HR əməkdaşı
- təlimə cavabdeh istifadəçi
- qiymətləndirmə ilə işləyən rəhbər və ya məsul şəxs

## Ən vacib məntiq
Performans nəticəsi sadəcə baxmaq üçün deyil. Əgər zəif qalan sahə varsa, onu təlim işi ilə davam etdirmək lazımdır.

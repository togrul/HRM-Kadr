# Leaves Ops / Commands Guide

Bu sənəd `Leaves` modulu üçün texniki və əməliyyat qeydlərini izah edir.

## 1. Modulun əməliyyat xarakteri

Leaves modulunun ayrıca geniş command səthi yoxdur. Əsas əməliyyatlar UI və observer/sync axını üzərindən işləyir.

Ops baxımından vacib sahələr:

- file storage
- leave observer
- Attendance sync
- export axını

## 2. Nəyi nə vaxt yoxlamaq lazımdır

`Approve olundu, Attendance dəyişmədi`:

1. leave observer işləyibmi
2. related sync xətası varmı
3. Attendance tərəfdə recalc lazım olub-olmadığını yoxla

`Sənəd açılmır`:

1. public storage
2. file path
3. upload sonrası permission

## 3. Tövsiyə olunan texniki yoxlama

- leave create
- leave approve
- leave cancel
- leave delete / restore
- export

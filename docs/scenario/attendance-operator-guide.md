# Attendance Operator Guide

Bu sənəd attendance məlumatını gündəlik işləyən operator üçündür.

## Gündəlik istifadə olunan tablar

- Daily monitor
- Puantaj grid
- Manual entries
- Exceptions inbox
- Overtime board

## Daily monitor

- bugünkü vəziyyəti tez görmək üçün istifadə olunur
- present / late / absent / missing ledger sayı buradadır
- search və structure scope birlikdə işləyir

## Manual entries

1. əvvəl structure seç
2. personnel seç
3. tarix və giriş/çıxış saatlarını daxil et
4. lazım olarsa explicit shift seç
5. manual override yalnız xüsusi hallarda aç
6. səbəb yaz
7. save et

## Exceptions inbox

- missing in/out və unmatched punch halları burada görünür
- həll edildikdə resolve et
- yanlış bağlanıbsa reopen et

## Overtime board

- overtime request-lər burada approve/reject olunur
- lazım olsa təsdiqlənən dəqiqəni düzəltmək olar

## Qaydalar

- səbəbsiz manual override etmə
- structure filter-i dəyişəndə nəticələrin scope-a uyğun olduğunu yoxla
- month locked olduqdan sonra dəyişiklik edilə bilməyəcəyini unutma

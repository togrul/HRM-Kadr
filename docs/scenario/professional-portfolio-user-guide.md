# Peşəkar portfel istifadəçi bələdçisi

## Bu modul nə üçündür?
Bu bölmə əməkdaşın peşəkar fəaliyyətlərini toplamaq və izləmək üçündür.

## Harada açılır?
Əməkdaş kartı içində `Peşəkar portfel` bölməsi açılır.

## Bu modulda nə saxlanılır?
- iştirak etdiyi vacib tədbirlər
- mediada görünən çıxışlar
- iştirak etdiyi layihələr

## Əsas bölmələr

### Tədbirlər
Konfrans, seminar və digər vacib tədbirlər burada saxlanılır.

### Media
Mediada çıxan xəbərlər, müsahibələr və digər görünüşlər burada görünür.

### Layihələr
Əməkdaşın iştirak etdiyi layihələr burada qeyd olunur.

### Zaman xətti
Bütün vacib qeydlər bir yerdə tarix sırası ilə görünür.

### Analitika
Ümumi göstəricilər və qısa hesabatlar burada görünür.

## Tipik istifadə qaydası

### Tədbir əlavə etmək
1. Tədbirlər bölməsinə keçin.
2. Tədbirin adını və tarixini yazın.
3. Rolu seçin.
4. Lazımdırsa sənəd və ya link əlavə edin.
5. Yadda saxlayın.

### Media qeydi əlavə etmək
1. Media bölməsinə keçin.
2. Başlıq və mənbəni yazın.
3. Link və ya arxiv faylı əlavə edin.
4. Yadda saxlayın.

### Layihə əlavə etmək
1. Layihələr bölməsinə keçin.
2. Layihənin adını, rolunuzu və qısa nəticəni yazın.
3. Yadda saxlayın.

### Yadda saxladıqdan sonra nə olur?
- qeyd siyahıda görünür
- lazım olarsa təsdiq gözləyə bilər
- təsdiqdən sonra ümumi görünüşdə və zaman xəttində görünə bilər

## Nəyə diqqət etmək lazımdır?
- Hər fəaliyyəti daxil etməyə ehtiyac yoxdur, vacib olanları yazın.
- Mümkündürsə sübut kimi sənəd və ya link əlavə edin.
- Təsdiq gözləyən qeydlər dərhal ümumi görünüşdə görünməyə bilər.

# Bildirişlər modulu bələdçisi

## Bu modul nə üçündür?
Bu modul bildirişləri hazırlamaq, göndərmək və izləmək üçündür.

## Harada açılır?
`Services > Bildirişlər` bölməsini açın.

## Bu modulda nə edə bilərsiniz?
- bildiriş mətnləri hazırlaya bilərsiniz
- qaydaları qura bilərsiniz
- elan göndərə bilərsiniz
- təsdiq gözləyən bildirişləri görə bilərsiniz
- göndərilən bildirişlərin tarixçəsinə baxa bilərsiniz

## Əsas bölmələr

### Ümumi baxış
Modulun qısa vəziyyətini göstərir.

### Şablonlar
Bildirişin mətnini burada yazırsınız.

### Qaydalar
Bildirişin kimə gedəcəyini burada seçirsiniz.

### Elanlar
Əl ilə elan yaratmaq üçündür.

### Göndərişlər
Göndərilmiş və ya gözləyən bildirişləri göstərir.

### Təsdiq növbəsi
Təsdiq tələb edən bildirişlər burada gözləyir.

### Tarixçə
Əvvəl göndərilmiş bildirişlərə baxmaq üçündür.

### Analitika
Ümumi göstəricilər burada görünür.

## Tipik istifadə qaydası

### Yeni şablon yaratmaq
1. Şablonlar bölməsinə keçin.
2. Mövzu və mətni yazın.
3. Yadda saxlayın.

### Qayda yaratmaq
1. Qaydalar bölməsinə keçin.
2. Bildirişin hansı halda gedəcəyini seçin.
3. Kimə gedəcəyini seçin.
4. Lazımdırsa təsdiq tələbini aktiv edin.
5. Qaydanı yadda saxlayın.

### Əl ilə elan göndərmək
1. Elanlar bölməsinə keçin.
2. Başlıq və mətni yazın.
3. Hədəf qrupu seçin.
4. Göndərin və ya planlaşdırın.

### Göndərdikdən sonra nə olur?
- elan göndərişlər siyahısında görünür
- təsdiq tələb olunursa əvvəl növbəyə düşür
- uğurla göndərilib-göndərilmədiyini tarixçədən görmək olur

## Nəyə diqqət etmək lazımdır?
- Mətnləri qısa və aydın yazın.
- Yanlış qrupa bildiriş getməməsi üçün seçimləri yoxlayın.
- Təsdiq tələb edən elanlar əvvəl növbəyə düşür, sonra göndərilir.

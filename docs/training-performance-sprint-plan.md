# Training + Performance Sprint Plan

## Sprint 1: Training Needs Foundation
- Training Needs module skeleton
- competency groups and levels
- competency catalog
- training program catalog
- program to competency mapping
- role competency requirement matrix
- foundation tables for employee competency profiles and training need items

## Sprint 2: Training Need Capture
- employee competency profile CRUD
- manual training need capture
- individual development plan shell
- priority and source handling
- status, target level, and target completion tracking

## Sprint 3: Performance Evaluation Foundation
- evaluation cycles
- evaluation template catalog
- template sections and weighted criteria items
- evaluator roles
- self, manager, and HR evaluation workflow
- weak score to training need bootstrap link

## Sprint 4: Skill Measurement
- test bank
- question bank and options
- test session assignment
- attempts
- auto-scored question types
- manual review for open answers
- skill profile refresh
- weak-area extraction

## Sprint 5: Integration and Planning
- performance gap to training need generation
- annual training plan board
- coverage and gap analytics
- reporting and documentation

### Delivered
- annual training plan creation
- approved/planned need aggregation into plan items
- coverage, mapping, and requirement-fit analytics
- source mix, priority mix, and top gap position reporting
- final training + performance user guide

# Candidates Module

- Namespace: `App\Modules\Candidates\Livewire`
- Views namespace: `candidates::`
- Routes: `app/Modules/Candidates/Routes/web.php`
- Provider: `App\Modules\Candidates\Providers\CandidatesServiceProvider` (loads routes/views, registers Livewire aliases).

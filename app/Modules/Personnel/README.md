# Personnel Module

Livewire components, routes, and views for personnel management.

- Namespace: `App\Modules\Personnel\Livewire`
- Views namespace: `personnel::`
- Routes: `app/Modules/Personnel/Routes/web.php` (home) and `print.php`
- Provider: `App\Modules\Personnel\Providers\PersonnelServiceProvider` (loads routes/views, registers Livewire aliases).
